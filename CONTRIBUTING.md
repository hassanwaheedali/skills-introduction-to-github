# Contributing to Introduction to GitHub

Thank you for your interest in learning GitHub! This repository is a demo project designed for educational purposes.

## About This Repository

This is a GitHub Skills course that teaches the fundamentals of using GitHub. It's designed to be completed in less than one hour and covers:

- Creating branches
- Making commits
- Opening pull requests
- Merging pull requests

## How to Use This Repository

1. Follow the instructions in the README.md file
2. Complete each step in sequence (Steps 0-4)
3. GitHub Actions will automatically guide you through the course
4. Each step builds on the previous one

## Getting Help

If you encounter any issues or have questions:

- Review the [GitHub Docs](https://docs.github.com)
- Ask questions in the [GitHub Skills Discussions](https://github.com/orgs/skills/discussions/categories/introduction-to-github)
- Check the [GitHub Status Page](https://www.githubstatus.com/) for platform issues
