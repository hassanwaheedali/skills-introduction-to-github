# Examples

This directory contains example files to help you learn GitHub basics.

## Files in This Directory

- **sample-profile.md** - An example GitHub profile README with common sections and formatting
  - Shows how to use Markdown formatting
  - Demonstrates badges and emojis
  - Provides a template you can customize

## How to Use These Examples

1. View the raw Markdown to see how formatting is done
2. Copy sections you like into your own PROFILE.md
3. Customize the content to reflect your own interests and goals
4. Experiment with different Markdown features

## Learn More About Markdown

- [Basic writing and formatting syntax](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)
- [GitHub Flavored Markdown Spec](https://github.github.com/gfm/)
- [Markdown Guide](https://www.markdownguide.org/)

## Tips for Creating Great Profile READMEs

- Keep it concise and easy to read
- Use emojis sparingly for visual interest
- Include links to your projects
- Showcase your skills and interests
- Keep it updated as you learn new things
